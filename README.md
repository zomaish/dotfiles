# Dotfiles (Nicolas Gallagher)

My OS X dotfiles.


## How to install

1. Clone into ~/.dotfiles

        $ git clone https://github.dev.bloomberg.com/eherdzi1/dotfiles.git ~/.dotfiles


2. Execute dotfiles script

        $ ~/.dotfiles/bin/dotfiles
